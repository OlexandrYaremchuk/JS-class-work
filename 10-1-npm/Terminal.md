```js
pwd - показує поточну дерикторію в якій ти знаходишся
ls  - показує всі файли в дерикторії в якій ви знаходитесь
ls -l  - показує всі файли списком
ls -la - опказує всі приховані файли
clear - очистка терміналу
cd - навігація по дерикторії
cd .. - перехід в папку по дерикторії вище
сd practice - перехід в конкретну папку
cd ../../ - перехід на два рівня вверх
touch test.md - створення файла в вибраній дерикторії
mv test.md test01.md - переіменування, перенесення файлу
rm test01.md - видалення вказаного файлу
rm css style.css - видалення файлу в вказаній папці
mkdir folderName -  створення папки
mkdir css/global  - папка в папці
mkdir css/global.main.css
man ls - дає повну тех дохументацію по цій команді(для виходу q)

Node.js

node -v - версія ноди
npm init -  ррозвертання пакетного менеджера npm в середині проекту
npm init -y (--yes) - автоматичне розгортання пакетного менеджера в проекті
npm install  - встановлення пакету npm

Бібліотеки
joi - бібліотека для валіації
uuid - бібліотеа для генерації унікального ідентифікатора
npm remove uuid - видалення пакета
shortid - бібліотка для генерації коротких унакальних id
```
